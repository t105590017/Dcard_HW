## 環境
||版本|
|-|-|

|npm|v6.14.11|
|OS|Windows10|

## 啟動
### 1. 本地啟動
```
git clone https://github.com/t105590017/Dcard_HW.git
cd Dcard_HW
npm install
npm start
```
